# Rushline: Highway Heat

Mobile-first arcade traffic racing game inspired by Traffic Racer.

## Play

Open `index.html` in a browser.

## Features

- Endless Drive, Time Attack, and Near Miss Challenge modes
- Three cars with different speed, acceleration, handling, braking, and boost values
- Persistent garage progress with banked coins, best score, and per-car upgrades
- Lane-based mobile controls with swipe, touch buttons, and keyboard support
- Traffic spawning, lane changes, collision detection, near-miss scoring, overtakes, boost, and Heat Chain multiplier
- Collectible coins and shield pickups during runs
- Pause/resume, simple sound toggle, countdown start, and improved result screen
- Responsive canvas layout for phones and desktop browsers

## Controls

- Swipe left/right or press the side buttons to change lanes
- Hold Brake to slow down
- Hold Boost to spend boost energy
- Pause button or `P`/`Esc` to pause
- Keyboard: Arrow keys or WASD, Space for boost

## Garage

- Earn coins from score, distance, near misses, collected coins, and completed objectives
- Spend banked coins on Engine, Turbo, Tires, Brakes, and Battery upgrades
- New saves start with 120 bank coins so the first upgrade is available immediately
- Progress is saved locally in the browser
